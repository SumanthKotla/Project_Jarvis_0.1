# Project_Jarvis_0.1
PROJECT JARVIS / FRIDAY — AI Voice Assistant
A real-time AI voice assistant inspired by Friday from the Iron Man universe, built with LiveKit Agents. Friday listens, thinks, and talks back — with a hint of sarcasm.
🧠 Stack

LLM — Groq (Llama 3.3 70B) for fast, free inference
STT — Deepgram for real-time speech recognition
TTS — Google Cloud Text-to-Speech (Journey voice)
VAD — Silero for voice activity detection

⚡ Features

🎙️ Real-time voice conversation
🎵 Play songs on YouTube by voice command
🌤️ Live weather with clothing advice
📧 Send emails by voice
🔊 System volume control
😴 Sleep/wake commands ("Hey Friday" / "Friday go offline")
🎭 Sarcastic personality with JARVIS-style responses
